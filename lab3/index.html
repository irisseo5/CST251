Challenge

Challenges I made is resizing the image. The way I use in this lab was to set the width and height myself. But there's more ways to resize images. 1. To resize the height automatically: img src="blah.jpg" style="width: (the size I want); height: auto;" 2. To make the image resize automatically : img src="blah.jpg" style="width: 100%; height: auto;" 3. img src="blah.jpg" style="max-width: 100%; height: auto;", put max- is optional. To reduce the size, we can also use different numbers instead of 100%
